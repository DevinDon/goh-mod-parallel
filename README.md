# Gate of Hell Mod - Parallel

地狱之门东线模组 - 平行时空，调整阵营强度，增加对抗烈度，新增新阵营。

自用，未上创意工坊。

## 准备工作

在 `C:\Program Files (x86)\Steam\steamapps\common\Call to Arms - Gates of Hell\mods` 目录下拉取仓库代码。

安装 [Node.js](https://nodejs.org/) 22+ 版本，使用 `corepack use pnpm` 启用 pnpm，然后运行 `pnpm install` 安装依赖。

## 构建模组

运行 `pnpm build` 构建模组，构建产物会自动打包并在 `C:\Program Files (x86)\Steam\steamapps\common\Call to Arms - Gates of Hell\mods\parallel` 目录生成 mod 文件夹。

详细的打包流程见 [src/packager/packager.mts](src/packager/packager.mts) 源文件。
